# 서울대학교 기숙사 웹 커뮤니티 'DOMITORI'
도미토리는 서울대학교 관악사 거주학생들을 위한 커뮤니티 웹 서비스 입니다. 기숙사 내 민원, 중고 거래 등과 관련된 게시글을 올리고 소통할 수 있습니다.
